# go-cosmos
Go driver for Azure CosmosDB SQL API

Features:

- Databases:
  - [x] CREATE DATABASE
  - [x] DROP DATABASE
  - [ ] LIST DATABASES
- Tables/Collections
  - [ ] CREATE TABLE/COLLECTION
  - [ ] DROP TABLE/COLLECTION
  - [ ] LIST TABLES/COLLECTIONS
- Items:
  - [ ] Azure CosmosDB SQL to INSERT/UPDATE/SELECT/DELETE items.
