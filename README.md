# go-cosmos
Go driver for Azure CosmosDB SQL API

Features:

- Databases:
  - [x] CREATE DATABASE
  - [x] DROP DATABASE
  - [x] LIST DATABASES
- Tables/Collections
  - [x] CREATE TABLE/COLLECTION
  - [x] DROP TABLE/COLLECTION
  - [x] LIST TABLES/COLLECTIONS
- Items:
  - [ ] Azure CosmosDB SQL to INSERT/UPDATE/SELECT/DELETE items.
